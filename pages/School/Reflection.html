Reflection

This is a reflection on my experience making this website as a class project

As far as progress goes towards the web page, I feel it almost never meets my satisfaction and that it will never be finished. I keep telling myself to tweak one more thing or spend loads of time configuring color combinations. Even now after submitting the link to the website at 10:45 the day before it is due, I'm sure that the web page will be drastically different by the time the grader views it. I'm content with the page layout. I want to add a bar at the bottom to traverse to different projects because right now u just must use the back arrow and then click on another assignment which I know can be inconvenient. I used a jet brain monospace font which is my favorite font and an available nerd font which I use on my Linux configuration across the board on my laptop, so I am confident in my font choice. One of the resources I used the most when creating and hosting my own website on GitHub pages was w3 schools because its hard to remember all the tags and code for html sometimes and it definitely helps to have a resource cheat seat on my second monitor.
